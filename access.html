Accessing Radicle

Radicle is a secure, peer-to-peer protocol for code collaboration - and it’s open-sourced. The protocol extends the open source software Git with peer-to-peer network discovery, disseminating software code and Git data via a process called “gossip”. Radicle is completely independent from Drips. At the moment, Radicle can be accessed via a command line interface (CLI) and web-app.

Accessing Drips

Drips is a decentralized toolkit for individuals and projects to support their software dependencies. The core component of Drips is an open-sourced, Ethereum-based protocol, which allows you to schedule and structure ERC-20 token transactions to particular addresses. There are various ways to interact with Drips’ protocol, such as those interfaces and access points linked here.
